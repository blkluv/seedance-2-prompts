## 🤔 什么是 Seedance 2.0？

**Seedance 2.0** 是**字节跳动**开发的视频生成模型，是业内首个支持**四模态同时输入**的模型——图片、视频、音频和文本。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐

#### 📝 提示词

```
场景 1：骏马驰骋 + 人群热情涌来 (0:00-0:08, 8 秒)

• 环境：细雪飘落，豪宅飞速掠过，远方是北岸山脉和海洋的轮廓
• 人群：前方突然出现一大群热情支持者（多元化年轻人），挥舞着黄色龙旗 + “Nothing” 标语，高呼呐喊，向骑手涌来
• 镜头：中速正面跟拍 → 转为骑手视角（POV），人群越来越近
• 气氛：高潮迭起，骑手自信微笑，准备拉缰绳（手已抬向缰绳，姿态放松，嘴角上扬）
场景 2：反转——人群径直穿过，无视骑手 (0:08-0:14, 6 秒)
• 地点：同一条路，转角处出现一家中式包子铺（招牌“庆丰包子铺”，暖色灯光，室外区域有简易塑料桌椅，热气腾腾）

• 动作：骑手笑容凝固，手停在半空，表情从期待变为震惊/困惑（风吹乱头发，西装下摆飘动）

乙（若有所思，嚼着包子，望向远方）：Nothing。

```
① 通用公式

主体 + 动作 + 场景 + 光线 + 镜头语言 + 风格 + 质量 + 约束

示例：

不要只写：走路 / 跳舞
要写：缓慢 + 持续 + 自然

优先词：
缓慢、轻柔、自然、持续、流畅、不僵硬

好用的组合：
缓慢转身 / 轻柔抬手 / 微微低头 / 随风摇曳

Seedance 在识别镜头运动上很强，记得写：

面部清晰，五官稳定，无畸形
人体结构正常，比例自然
相同人物，服装一致，发型不变
