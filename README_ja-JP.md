## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

```

• 被写体：ダークスーツを着た人物が馬に乗り、邸宅ゲートから疾走してくる
• 馬の動き：白馬の四肢が空中に浮き、たてがみがなびき、速いペースまたは速足で駆け抜ける

• カメラ：正面からのショット、白馬がカメラに向かって直接突進し、強い動きの感覚（インパクトを高めるためにわずかにローアングル）
